Operations Manual

1. Server setup

Install Server

In the desired directory, run git clone https://github.com/Imagifight/election.
This will copy the required files into the directory election.
Within election, run npm i and wait for installation to complete.

Start MongoDB at localhost:27017. Create database elect with collections voters and candidates.
Create a CSV file for the voters with fields name, grade, sec, house called voters.csv and place it in the folder csv. Execute the Python script CSVtoJSON.voters.py.
The voters have been inserted into the database.

Start Sever

Run npm start in the root directory (elections). Wait for the console to display Starting the server at 0.0.0.0:8080.
In a browser open localhost:8080/candidates/upload and fill in the form to add candidates. Repeat as many times as required.

Note: currently only accepts .jpg files.

2. Connect Terminal

On the host computer run ipconfig and get the host IP address. On the computer used for collecting votes, type in ip_address:8080 in a browser where ip_address is the host IP.
Ensure the screen reads lock locked
On the host computer open localhost:8080/console and select the required class. Elections may now begin.

3. Results

After voting ends, open localhost:8080/tally to see results.
Clear the MongoDB database.
